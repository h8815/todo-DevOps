About MyTodo App

Welcome to MyTodo, your minimal, clean, and modern to-do list manager built with 💖 using Flask and Bootstrap.

📌 Why this app?

MyTodo is designed to help you stay organized, focused, and productive. Whether you're managing personal tasks, keeping track of work, or practicing your web development skills — this app has your back.

🛠️ Tech Stack

  • Flask – Lightweight Python web framework
  • SQLAlchemy – ORM for interacting with the database
  • SQLite – Simple and fast relational database
  • Bootstrap 5 – Beautiful responsive UI
  • Jinja2 – Template engine for rendering dynamic pages

✨ Features

  • ✔️ Add, update, and delete todos
  • ✔️ Clean user interface with responsive design
  • ✔️ Smooth animations and hover effects
  • ✔️ Timestamped todos to keep you on track
  • ✔️ Built for both practice and real use

🚀 Want to Build More?

This project is a great foundation to add more features like user login, categories, dark mode, or cloud storage.
